# vi-cheatsheet
Just a bunch of vi text editor commands <br>
> New file creation : vi [name of the file] <br>
>> Example : To create a file named science.txt, type **vi science.txt** <br>
>> This creates the file and the we enter inside the file. <br>

> There are two modes in vi text editor : command mode and insert/input mode <br>
