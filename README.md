# vi-cheatsheet
Just a bunch of vi text editor commands <br>
> New file creation : vi *file name* <br>
>> Example : To create a file named science.txt, type **vi science.txt** <br>
>> This creates the file and then we enter inside the file. <br>

> There are two modes in vi text editor : command mode and insert/input mode <br>
>> command mode : helps in navigation and execution related commands like saving, renaming ect,.. <br>
>> insert/input mode : text can be added into the file <br>

> Vi always starts with command mode. To go to insert mode use the following commands below <br>
>> 1. a - append character after cursor <br>
>> 2. i - insert before cursor <br>
>> 3. o - open line below <br>
